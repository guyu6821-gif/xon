## 🚀 Xüsusiyyətlər

### 1️⃣ Semestr Bal Hesablama

**Necə istifadə olunur:**

1. Ana səhifədə **"Semestr Bal Hesablama"** kartuşuna basın
2. **Seminar sayı**nı daxil edin (maksimum 9)
3. Hər seminar üçün **balınızı** yazın (0-10 arası)
4. **Kollekvium sayı**nı daxil edin (maksimum 4)
5. Hər kollekvium üçün **balınızı** yazın (0-10 arası)
6. **Sərbəst iş balı**nızı yazın (0-10 arası)
7. **Dərs saatı**nızı seçin (30/45/60/75/90/105)
8. **Qayıb sayı**nızı yazın
9. **"Hesabla"** düyməsinə basın
10. Nəticənizi görün! 🎉

**Qiymətləndirmə:**
- 50+ bal: 🎉 MÜVƏFFƏQİYYƏTLƏ KEÇDİNİZ! ✅
- 45-49 bal: 🔥 ÇOX YAXŞI 📊
- 41-44 bal: 💣 YAXŞI 📈
- 36-40 bal: 🫂 KAFİ 📉
- 26-35 bal: 🎭 ZƏİF 📴
- 0-25 bal: 🗿 YAXŞI OLACAQ 🆒

### 2️⃣ ÜOMG Hesablama

**Necə istifadə olunur:**

1. **"ÜOMG Hesablama"** kartuşuna basın
2. **Fənn sayı**nı yazın (neçə fənniniz varsa)
3. **"Sahələr Yarat"** düyməsinə basın
4. Hər fənn üçün:
   - **Balınızı** yazın (0-100 arası)
   - **Kredit sayı**nı yazın
5. **"Hesabla"** düyməsinə basın
6. ÜOMG nəticənizi görün!

**Düstur:** (bal₁×kredit₁ + bal₂×kredit₂ + ... + balₙ×kreditₙ) / (kredit₁ + kredit₂ + ... + kreditₙ)

### 3️⃣ 25% İmtahan Pulu (Kəsr Pulu)

**Necə istifadə olunur:**

1. **"25% İmtahan Pulu"** kartuşuna basın
2. **İllik ödəniş**inizi yazın (AZN-lə)
3. **Fənnin kredit sayı**nı yazın
4. **"Hesabla"** düyməsinə basın
5. Ödəməli olduğunuz məbləği görün

**Düstur:** ((illik ödəniş / 60) × kredit sayı) / 4 + 1

### 4️⃣ Yaş Hesablayıcı

**Necə istifadə olunur:**

1. **"Yaş Hesablayıcı"** kartuşuna basın
2. **Doğum tarixinizi** seçin
3. **"Hesabla"** düyməsinə basın
4. Görəcəksiniz:
   - Yaşınız (il, ay, gün)
   - Yaşadığınız günlərin sayı
   - Növbəti ad gününüzə qalan günlər
